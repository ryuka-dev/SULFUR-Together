> [b]Version 1.1.0 — Public Beta: SULFUR 0.18.3 support, shared loot, and fixes.[/b] It works, but it is still being polished. Expect bugs, back up your saves, and make sure everyone runs the same version.

[color=RoyalBlue][b]English[/b][/color]

[b]SULFUR Together[/b] lets you play SULFUR co-op. Explore the same procedurally generated levels, fight enemies together, revive downed teammates, see each other's weapons and attacks, break objects in the world, and drop items for each other.

You connect [b]inside the game[/b] now — there is a built-in menu for hosting and joining over [b]Direct IP[/b] or [b]Steam[/b]. No config-file editing.

[color=DodgerBlue][b]⚠️ Public Beta notice[/b][/color]

This is a [b]public test build[/b]. The main loop works from start to finish, but many systems are still being optimized and there is a lot left to refine.

[list]
[*]Bugs, visual desyncs, and the occasional broken boss or level transition can happen.[/*]
[*][b]Back up your save files[/b] before playing.[/*]
[*]Every player must use the [b]same version of SULFUR[/b] and the [b]same version of SULFUR Together[/b].[/*]
[/list]
This release is for players who are happy to test unfinished multiplayer and report problems.

[color=DodgerBlue][b]Requirements[/b][/color]

[list]
[*][b]SULFUR[/b] (Steam)[/*]
[*][b]BepInEx 5[/b] — installed automatically as a dependency[/*]
[*][b]SULFUR Native UI Lib 0.10.1[/b] — installed automatically as a dependency. It powers the in-game connect menu, notifications, and translations.[/*]
[/list]
You do [b]not[/b] have to install these by hand if you use a mod manager (below).

[color=DodgerBlue][b]How to install (recommended: Gale)[/b][/color]

[b]Gale[/b] is a free, beginner-friendly mod manager. If you have never modded a game before, use this.

[list=1]
[*]Download [b]Gale[/b] from its official page: [b]https://github.com/Kesomannen/gale[/b] — open the [b]Releases[/b] section and grab the latest installer for your system.[/*]
[*]Install and open Gale. When it asks which game to manage, choose [b]SULFUR[/b].[/*]
[*]Open the [b]Browse mods[/b] tab and search for [b]SULFUR Together[/b].[/*]
[*]Click [b]Install[/b]. Gale automatically pulls in everything it needs (BepInEx and SULFUR Native UI Lib) — you do not need to install those separately.[/*]
[*]Press the [b]Launch game (modded)[/b] button inside Gale. The first launch takes a little longer while BepInEx sets itself up.[/*]
[*]Every player who wants to join must do the same steps and install the [b]same version[/b].[/*]
[/list]
That's it — you're ready to host or join from inside the game.

[color=DodgerBlue][b]How to connect[/b][/color]

Co-op is hosted and joined [b]from inside a loaded save[/b] (not from the title screen).

[list=1]
[*]Load into your game (the hub or any level).[/*]
[*]Open the [b]Options[/b] menu and go to the [b]SULFUR Together[/b] page.[/*]
[*]Set your [b]Player name[/b].[/*]
[*][b]To host:[/b] press [b]Create game[/b]. Then either:[/*]
[/list]
[list]
[*][b]Steam:[/b] press [b]Invite Friends via Steam[/b] — no port forwarding needed; or[/*]
[*][b]Direct IP:[/b] share the [b]LAN address[/b] shown on the page (and, over the internet, forward the UDP port or use a VPN LAN). Everyone must use the same [b]Connection key[/b].[/*]
[/list]
5. [b]To join:[/b]
[list]
[*][b]Steam:[/b] accept the invite (it joins automatically once you load a save), or paste the host's [b]Steam ID[/b] and press [b]Join via Steam[/b]; or[/*]
[*][b]Direct IP:[/b] enter the host's [b]address[/b], [b]port[/b], and [b]connection key[/b], then press [b]Join game[/b].[/*]
[/list]
6. Use [b]Close room[/b] (host) or [b]Leave[/b] (client) to end the session.

Your settings save automatically as you type. Quick keys still exist: [b]Page Down[/b] links/follows the host, [b]Page Up[/b] unlinks and returns you to solo play.

[color=DodgerBlue][b]Saves, inventory, and loot[/b][/color]

SULFUR Together does [b]not[/b] use a shared character save. Each player keeps their own character, inventory, equipment, progression, personal loot, and save data. Items you deliberately drop into the world can be picked up by others. [b]Optional shared loot[/b] is available too — the host can turn on [i]Shared loot[/i] on the connect page, which shares enemy, crate, chest, and hatbox/register loot (first player to grab each item takes it). It is [b]off by default[/b]; otherwise each player gets their own loot as before.

[color=DodgerBlue][b]No-pause multiplayer[/b][/color]

The world does [b]not pause[/b] while multiplayer is active — opening the inventory, the pause menu, dialogue, or moving the window out of focus does not stop the game. This is intentional, so players don't desync. [b]Get to a safe spot before opening menus.[/b]

[color=DodgerBlue][b]Known issues[/b][/color]

[list]
[*]A client that loads a boss level ahead of the host can desync (e.g. Cousin "infinite dialogue").[/*]
[*]Some enemies may briefly snap, teleport, or animate incorrectly.[/*]
[*]Bosses beyond the tested set may have serious sync problems.[/*]
[*]Blood, particles, sounds, and some animations are not always synchronized.[/*]
[*]Player collision is experimental; remote sprites/weapons are still being refined.[/*]
[*]Sessions with 4+ players are untested — 2 players is the most predictable.[/*]
[/list]
This list is not exhaustive.

[color=DodgerBlue][b]存档、背包与掉落[/b][/color]

SULFUR Together [b]不使用[/b]共享角色存档。每位玩家保留自己的角色、背包、装备、进度、个人掉落与存档数据。你[b]主动[/b]丢到世界里的物品可以被其他人捡起。现在还提供[b]可选的共享掉落[/b]——房主可在连接页开启 [i]Shared loot（共享掉落）[/i]，此时敌人、木箱、宝箱以及帽盒/收银机的战利品都会共享（谁先捡到归谁）。该功能[b]默认关闭[/b]；关闭时每位玩家和以前一样获得各自的掉落。

[color=DodgerBlue][b]联机不暂停[/b][/color]

联机进行时游戏世界[b]不会暂停[/b]——打开背包、暂停菜单、对话或把窗口切到后台都不会让游戏停下。这是[b]有意为之[/b]，以避免玩家之间不同步。[b]打开菜单前请先到安全的位置。[/b]

[color=DodgerBlue][b]セーブ・インベントリ・戦利品[/b][/color]

SULFUR Together は共有キャラクターセーブを[b]使いません[/b]。各プレイヤーは自分のキャラクター・インベントリ・装備・進行・個人戦利品・セーブデータを保持します。あなたが[b]意図的に[/b]ワールドへ落としたアイテムは他のプレイヤーが拾えます。さらに[b]任意の共有戦利品[/b]も利用できます——ホストが接続ページで [i]Shared loot（共有戦利品）[/i] をオンにすると、敵・木箱・宝箱・帽子箱／レジの戦利品が共有されます（各アイテムを最初に拾った人が入手）。[b]既定ではオフ[/b]で、オフの場合は従来どおり各プレイヤーが自分の戦利品を得ます。

[color=DodgerBlue][b]一時停止しないマルチプレイ[/b][/color]

マルチプレイ中はゲーム世界が[b]停止しません[/b]——インベントリ、ポーズメニュー、会話を開いても、ウィンドウを非アクティブにしてもゲームは止まりません。これはプレイヤー間のずれを防ぐための[b]意図的な仕様[/b]です。[b]メニューを開く前に安全な場所へ移動してください。[/b]
